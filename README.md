# Dotfiles

My basic dotfiles configuration.

To install and run the script, run the following:
```bash
wget -O - https://raw.githubusercontent.com/StephanMalan/dotfiles/master/install.sh | bash
```

Installs:
- Basic tools
- Docker
- VS Code
- Intellij
- Rust & useful crates
- Oh-my-zsh, plugins and theme
- Alacritty
- Zellij
